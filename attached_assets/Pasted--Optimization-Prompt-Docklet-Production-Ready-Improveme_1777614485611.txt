* Each container has isolated env variables (encrypted at rest - AES-256)

### Apply & Restart (Core Logic)

When env changes:

1. Use `docker inspect` to fetch full container config
2. Merge updated env variables
3. Perform:

   * Stop container
   * Remove container
   * Recreate container with updated env
4. Preserve:

   * Volumes
   * Networks
   * Labels
   * Ports

## 🌐 3. Domain System (Upgrade to Traefik) 🔥

Replace nginx static routing with dynamic routing in Doamin genarte using:

👉 Traefik

---

### Traefik Integration

* Auto-detect containers using Docker labels
* No manual config file writing
* No nginx reload required

### Example Container Labels:

```yaml
labels:
  - "traefik.enable=true"
  - "traefik.http.routers.app.rule=Host(`app-abc.yourdomain.com`)"
  - "traefik.http.services.app.loadbalancer.server.port=3000"
```

---

### Features:

* Dynamic routing (zero config writes)
* Automatic HTTPS via Let's Encrypt
* Wildcard domain support
* Zero-downtime domain attach
* Auto domain generation:

  * `app-<random>.yourdomain.com`

---

### Domain Flow:

1. One-time DNS verification:

   * `A @ → VPS IP`
   * `A * → VPS IP`
2. Generate subdomain
3. Attach labels to container
4. Traefik auto routes
